# YouTube Frontend UI Clone 🎬

This project is a frontend clone of the YouTube user interface built using **HTML and CSS**.  
It was created as part of a practical assignment to improve frontend development skills and GitHub workflow.

---

## 📌 Project Objective
To design and implement a YouTube-style user interface using only HTML and CSS while maintaining a clean project structure and meaningful Git commits.

---

## 🚀 Features
- Responsive header with logo, search bar, and profile section
- Sidebar navigation menu
- Video grid layout
- Thumbnail previews with titles and channel info
- Clean and structured CSS styling
- Responsive design basics

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Git
- GitHub
